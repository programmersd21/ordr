# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0]

### Added
- Initial release of ordr
- Core sorting algorithms: bubble, insertion, merge, quick, heap, intro, tim, pdq, radix
- Adaptive smart sort with automatic algorithm selection
- Parallel sorting support via Rayon
- Python bindings via PyO3
- Rich CLI with info, bench, and visualize commands
- Comprehensive benchmarking suite
- Terminal-based sorting visualization
- Full documentation with MkDocs Material
- CI/CD pipeline with GitHub Actions
